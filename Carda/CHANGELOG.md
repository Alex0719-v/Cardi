# 更新日志

## 1.0.0 - 2026-07-20

- 完成 Cardi 品牌与工程命名迁移，保留 Carda 内部兼容标识以保护签名、Keychain、本地账户归档和附近交换兼容性。
- 完善账户体系、设置中心、关联应用、数据与存储、交互与辅助功能等本地管理能力。
- 重构名片交换流程，增加近距离目标选择、递卡/接卡/回递动画链路和模拟器回归测试入口。
- 强化名片夹、我的名片与底部导航的连续形变、列表模式隔离和整体视觉一致性。

## 0.6.0 - 2026-07-05

- 统一“我的名片”、名片夹、搜索与编辑页面的背景视觉，并完善页面切换、底部导航和分段控件的连续形变动画。
- 重构名片夹的姓名、公司与列表模式，增加折叠/展开、分组索引、左滑删除、列表管理及联系人操作弹窗。
- 优化名片创建与展示体验，完善动态字段、手机号格式化、键盘避让、账户面板、多名片分页和长按操作。
- 改进 5cm 近距离名片交换流程的测距判断、状态管理与连接稳定性。

## 0.3.0 - 2026-06-22

- 彻底更改了名片夹的样式，增加了名片夹的交互方式，更改了部分低效的交互逻辑。另外增加了交换名片的功能。

## 0.1.2 - 2026-06-17

- 更新了名片的设计样式，更符合 UI 的交互方式。
