# Cardi

Cardi 是一个 iOS 社交类名片软件，主要用于名片创建、名片交换与名片管理。

用户可见品牌、Xcode 工程、Target、Scheme、构建产品、Swift 模块和 App 入口均已更新为 `Cardi`。为保留签名、Keychain 和本地账户数据兼容性，Bundle ID、Application Support 的 `Carda/` 目录与既有偏好 Key 暂时沿用内部旧名称。

> 协作规则：开发前优先阅读 `AGENTS.md` 与 `contributing_ai.md`。

---

## 产品概览

Cardi 的核心对象是“名片”。用户可以创建自己的交互名片，在“我的名片”页面上划当前名片，递给 1.5 米内方向明确且最近的 Cardi 用户，并在名片夹中管理收到的名片。

主要页面：
- 我的名片
- 名片夹
- 名片搜索

主要能力：
- 创建一张或多张自己的名片。
- 编辑姓名、英文名/拼音、职位、公司、头像、公司 logo 和联系信息。
- 根据输入内容动态生成名片布局。
- 将名片保存为 `.png` 图片。
- 在名片夹中按姓名、分类、公司查看收到的名片。
- 搜索名片，按姓名、单位和其他信息分类展示结果。
- 单方上划递交当前名片，双方在短时间内上划时自动互换；单向来卡可翻转回递。

## 当前项目结构

- `CardiApp.swift`：应用入口与 SwiftData 容器。
- `ContentView.swift`：进入 `AppShellView`。
- `Item.swift`：模板遗留模型，暂保留在 SwiftData schema 中以避免迁移风险。
- `Models/`：名片、字段、名片列表、草稿和字段类型。
- `Views/MyCards/`：我的名片、创建/编辑名片。
- `Views/CardHolder/`：名片夹。
- `Views/Search/`：名片搜索。
- `Components/`：名片渲染、折叠名片、底部导航、头像等复用组件。
- `Components/LocalSVGIconView.swift`：本地 SVG 图标缓存解析与 SwiftUI `Canvas` 渲染，用于底部导航和名片右侧操作按钮等高频图标。
- `Components/TransparentGradientBlur.swift`：封装通用渐变毛玻璃，并提供名片夹专用的 `FigmaScrollEdgeSoftOverlay` 屏幕定位 Soft Edge fallback。
- `Services/`：保存名片图片、历史演示数据清理、默认列表补齐与近距离名片交换等服务。
- `Services/CardExchange/`：前台被动广播/上滑主动发现、Nearby Interaction token 测距与方向、目标选择、单/双向交换消息、持久化 ACK 和名片 payload。
- `DesignSystem/`：颜色、尺寸、玻璃样式和名片常量。
- `AppIcon.icon/`：Icon Composer 应用图标工程包及其本地 SVG 图层。
- `My icon/`：底部导航 tab、名片右侧操作按钮与名片夹删除按钮的 SVG 图标源文件。
- `Card photo/`：三张名片底图 PNG 资源；`Group 42.png` 对应 `color1`，另含 `color2.png`、`color3.png`。
- `Assets.xcassets/`：资源目录。
- `Info.plist`：本地网络、Bonjour、相册保存与应用基础 plist 配置。
- `ExchangeTestPlan.md`：交换功能上线测试矩阵、真机步骤、当前证据、风险分级与放行门槛。
- `../CardiLogicTests/`：可直接用 `swiftc` 编译的纯逻辑回归。
- `../CardiUITests/CardExchangeSimulationUITests.swift`：DEBUG Simulator 互换与单向翻面回递自动化。
- `AGENTS.md`：项目级 AI 指导规则。
- `contributing_ai.md`：AI 协作与变更协议。
- `Designsystem.md`：设计系统记录。
- `ROADMAP.md`：路线图与待确认事项。

## 运行方式

1. 使用 Xcode 打开项目。
2. 选择目标设备或模拟器。
3. Build & Run。

命令行构建：

```bash
xcodebuild -project ../Cardi.xcodeproj -scheme Cardi -destination 'generic/platform=iOS Simulator' build
```

交换 UI 模拟回归：

```bash
xcodebuild -project ../Cardi.xcodeproj -scheme Cardi \
  -destination 'platform=iOS Simulator,name=iPhone 17 Pro' \
  -only-testing:CardiUITests/CardExchangeSimulationUITests test
```

完整交换测试范围、真机准备和判定标准见 `ExchangeTestPlan.md`。
